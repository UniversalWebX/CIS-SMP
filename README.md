# Messaging Service

A simple real-time chat application using WebSockets.

## Files
- `server/server.py` - WebSocket server (deploy this on Render)
- `server/requirements.txt` - Python dependencies
- `client/index.html` - Frontend chat client (can be hosted anywhere)

## Deploy on Render
1. Create a new Web Service
2. Upload the `server` folder
3. Set start command: `python server.py`
4. Port is automatically detected (8000)

## Use the client
Open `client/index.html` in a browser and point it to your Render URL.

Enjoy!
